## Requirements
Node v18.14.0
NPM v9.6.4
OS Windows 11

## Installation
- Clone repository
- Run `make install` in the root folder
- Run `make start`
- Enjoy

## Description
The Slack analog SPA. This messager can be used for personal purposes. It has its backend and frontend parts in the same repository.
